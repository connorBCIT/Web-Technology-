Lab 1 - HTML basics

HTML, short for Hyper Text Markup Language is the standard language for creating web pages. HTML explains to a web browser how to display content. This is done using tags. For example this paragraph is wrapped in a center tag, centering the paragraph.


The default alignment for text in html is left. However, using a tag we can change the alignment. For this paragraph the "p" tag is attached with an attribute: 'p align ="right"'
